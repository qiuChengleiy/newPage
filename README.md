### 🌱 学习心得

* 坚持每天写代码(不管写啥)
* 坚持阅读(我经常会看一些比较好的公众号的推文，然后动手去实践)
* 多动手，多搜索，多思考（有时候问别人，不如自己动手去寻找答案，这个过程会让你学习到更多的知识）
* 端正观念，误听他言（不管社会多卷多，多现实，年龄多大，如果你还热爱这一行，那就大胆去做）
* 敢于尝试，没有敲不会的代码，只有偷懒的人
